# Task Manager App

A full-stack task manager built with React, Node.js, Express, and MongoDB.

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Environment Variables
Create a `.env` file in the `backend/` folder:
```
PORT=5000
MONGO_URI=mongodb://localhost:27017/taskmanager
```
